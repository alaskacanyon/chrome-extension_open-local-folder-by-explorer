Initialization

Those steps are required to allow the extension to be fully operational.

  1. Install Node.js.
    ※ The LTS recommended version is OK.
  2. Create a folder in a suitable location that will not be deleted.
    ※ If the folder disappears due to deletion or movement, this initial setup needs to be re-enforced.
  3. Save the following files in the folder you just created.
    • manifest.json
    • native-messaging-host-app.bat
    • native-messaging-host-app.js
